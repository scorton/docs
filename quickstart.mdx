## Welcome to Scorton

Scorton is an AI-powered educational platform that helps create personalized learning experiences. This quickstart guide will help you get up and running quickly.

## Getting Started

### 1. Sign Up for an Account

<AccordionGroup>
  <Accordion title="Create your account" icon="user-plus">
    Visit [scorton.tech](https://scorton.tech) and click "Sign Up" to create your account. You can sign up as an educator, student, or administrator.
  </Accordion>
  <Accordion title="Get your API key" icon="key">
    After signing up, navigate to your dashboard and go to the API Keys section. Generate a new API key to start integrating with our platform.
  </Accordion>
</AccordionGroup>

### 2. Set Up Your First Course

<AccordionGroup>
  <Accordion title="Create a course" icon="book">
    Use our course builder to create your first educational course. You can add lessons, quizzes, and interactive content.
  </Accordion>
  <Accordion title="Invite students" icon="users">
    Share your course with students by sending them invitation links or adding them directly through the dashboard.
  </Accordion>
</AccordionGroup>

## API Integration

### Quick API Test

Test your API key with a simple request:

```bash
curl -X GET "https://api.schoolyai.com/v1/courses" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### SDK Installation

Install our official SDK for your preferred language:

<CardGroup cols={2}>
  <Card title="JavaScript/TypeScript" icon="js" href="https://github.com/SchoolyAI/sdk-js">
    ```bash
    npm install @schoolyai/sdk
    ```
  </Card>
  <Card title="Python" icon="python" href="https://github.com/SchoolyAI/sdk-python">
    ```bash
    pip install schoolyai
    ```
  </Card>
</CardGroup>
